Hi, this is the first site I've written on my own.

Run from the index.html file.

The Photoshop layout of the site is located in the Source folder
The binary folder contains all the site files

Styles are written in scss
folder \ project name \ sourse \ scss \ style.scss
and converted by prepros to css file
folder \ project name \ css \ style.css

JavaScript is located in the \ project name \ js \ script.js folder

BEM methodology (block, element, modifier) was used.

Have a nice day! :)
